Java Documentation

1. Introduction to Java

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It runs on the Java Virtual Machine (JVM), which makes it platform independent.

Learn More on Oracle Docs

2. Setting up Java

  • Install JDK (OpenJDK or Oracle JDK)
  • Set JAVA_HOME and add bin to PATH
  • Use build tools: Maven or Gradle for projects

3. Core Concepts

  • Object-Oriented Programming: classes, objects, inheritance, polymorphism
  • Primitive types and reference types
  • Exception handling with try/catch/finally
  • Collections framework (List, Set, Map, Queue)

4. Concurrency

Java provides threading support via Thread and higher-level concurrency utilities in java.util.concurrent. 


// Simple thread example
public class HelloThread extends Thread {
    public void run() {
        System.out.println("Hello from a thread");
    }
}

5. Common APIs & Tools

  • Streams & Lambdas (Java 8+)
  • JDBC for database access
  • Build tools: Maven / Gradle
  • Testing: JUnit
