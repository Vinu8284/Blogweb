Node.js Documentation

1. Introduction to Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows developers to run JavaScript on the server side.

Learn More on Node.js Official Docs

2. Installation

To install Node.js, download it from the official website or use a package manager like `nvm`.

                # Using nvm
nvm install node
nvm use node

3. Basic Example

Here is a simple example of a Node.js HTTP server:

                const http = require('http');

const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello, World!');
});

server.listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

6. Express.js

Express.js is a popular web framework for Node.js.

                const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send('Hello, Express!');
});

app.listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

7. File Handling

Node.js provides the `fs` module for file handling.

                const fs = require('fs');

// Write to a file
fs.writeFileSync('example.txt', 'Hello, Node.js!');

// Read from a file
const data = fs.readFileSync('example.txt', 'utf8');
console.log(data);

8. Asynchronous Programming

Node.js uses asynchronous programming with callbacks, promises, and async/await.

                // Using Promises
const fs = require('fs').promises;

async function readFile() {
    try {
        const data = await fs.readFile('example.txt', 'utf8');
        console.log(data);
    } catch (err) {
        console.error(err);
    }
}

readFile();

12. Streams

Streams are used to handle reading or writing data in chunks, making them efficient for large files.

                    const fs = require('fs');
    
    const readStream = fs.createReadStream('largeFile.txt', 'utf8');
    readStream.on('data', chunk => {
        console.log(chunk);
    });

13. Cluster Module

The cluster module allows you to create child processes to utilize multiple CPU cores.

                    const cluster = require('cluster');
    const http = require('http');
    const os = require('os');
    
    if (cluster.isMaster) {
        const numCPUs = os.cpus().length;
        for (let i = 0; i < numCPUs; i++) {
            cluster.fork();
        }
    } else {
        http.createServer((req, res) => {
            res.writeHead(200);
            res.end('Hello, Cluster!');
        }).listen(8000);
    }
