JavaScript Documentation

1. Introduction to JavaScript

JavaScript is a high-level, interpreted programming language used to make web pages interactive. It is one of the core technologies of the web, alongside HTML and CSS.

Learn More on MDN

2. Variables

Variables are used to store data. JavaScript has three ways to declare variables: var, let, and const. 


let name = "John"; // Mutable variable
const age = 25; // Immutable variable
var isOld = false; // Avoid using var (legacy)
Learn More on MDN

3. Data Types

JavaScript has dynamic types. The most common data types are:

  • String: "Hello, World!"
  • Number: 42
  • Boolean: true or false
  • Array: [1, 2, 3]
  • Object: { name: "John", age: 25 }
Learn More on MDN

4. Functions

Functions are reusable blocks of code. They can be declared using the function keyword or as arrow functions. 


function greet(name) {
    return `Hello, ${name}!`;
}

// Arrow function
const greet = (name) => `Hello, ${name}!`;
Learn More on MDN

5. Loops

Loops are used to execute a block of code repeatedly. Common loops include for, while, and for...of. 


for (let i = 0; i < 5; i++) {
    console.log(i);
}

const numbers = [1, 2, 3];
for (const num of numbers) {
    console.log(num);
}
Learn More on MDN

6. Conditionals

Conditionals are used to execute code based on certain conditions. Common conditionals include if, else, and switch. 


const age = 18;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

The Document Object Model (DOM) allows JavaScript to interact with HTML elements.

8. ES6 Features

ES6 introduced many new features, including arrow functions, template literals, and destructuring. 


// Arrow function
const add = (a, b) => a + b;

// Template literals
const name = "John";
console.log(`Hello, ${name}!`);

// Destructuring
const person = { name: "John", age: 25 };
const { name, age } = person;

9. Async/Await

async and await make asynchronous code easier to write and read. 


async function fetchData() {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    console.log(data);
}

12. Modules

JavaScript modules allow you to split code into reusable pieces.


// math.js

// main.js
import { add } from "./math.js";

13. OOP Concepts

JavaScript supports object-oriented programming with classes and inheritance. 


class Animal {
    constructor(name) {
        this.name = name;
    }

    speak() {
        console.log(`${this.name} makes a noise.`);
    }
}

class Dog extends Animal {
    speak() {
        console.log(`${this.name} barks.`);
    }
}

const dog = new Dog("Rex");
dog.speak();

14. Advanced Topics

Closures

A closure is a function that retains access to its lexical scope, even when the function is executed outside that scope. 


function outer() {
    const message = "Hello";
    function inner() {
        console.log(message);
    }
    return inner;
}

const innerFunc = outer();
innerFunc(); // Logs "Hello"
Learn More on MDN

Prototypes

JavaScript uses prototypal inheritance, where objects can inherit properties and methods from other objects. 


const person = {
    greet() {
        console.log("Hello!");
    }
};

const john = Object.create(person);
john.greet(); // Logs "Hello!"
