CSS Documentation

1. Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall visual appearance of a webpage.

Learn More on MDN

2. Selectors

CSS selectors are used to target HTML elements for styling. Common selectors include:

  • Element Selector: p
  • Class Selector: .class-name
  • ID Selector: #id-name
  • Attribute Selector: [type="text"]
Learn More on MDN

3. Box Model

The CSS box model describes the structure of elements, including:

  • margin: Space outside the element.
  • border: Border around the element.
  • padding: Space inside the element, around the content.
  • content: The actual content of the element. 

.box {
    margin: 10px;
    border: 2px solid #007BFF;
    padding: 20px;
    width: 200px;
}
Learn More on MDN

4. Flexbox

Flexbox is a layout model that allows you to design flexible and responsive layouts. 


.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
Learn More on MDN

5. CSS Grid

CSS Grid is a two-dimensional layout system for creating complex grid-based designs. 


.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    gap: 10px;
}
Learn More on MDN

6. Animations

CSS animations allow you to animate transitions between styles.


@keyframes slide {
    0% { transform: translateX(0); }
    100% { transform: translateX(100px); }
}

.box {
    animation: slide 2s infinite;
}
Learn More on MDN

7. Transitions

CSS transitions enable smooth changes between property values.


.button {
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: #007BFF;
}
Learn More on MDN

8. Pseudo-classes

Pseudo-classes are used to define special states of elements.


a:hover {
    color: #FF69B4;
}

li:nth-child(odd) {
    background-color: #333;
}
Learn More on MDN

9. CSS Variables

CSS variables allow you to store reusable values.


:root {
    --primary-color: #007BFF;
}

.button {
    background-color: var(--primary-color);
}
Learn More on MDN

10. Responsive Design

Use media queries to create responsive layouts.


@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}
Learn More on MDN

11. CSS Preprocessors (SASS/LESS)

Preprocessors like SASS and LESS extend CSS with features like variables, nesting, and mixins. 


$primary-color: #007BFF;

.button {
    background-color: $primary-color;
}
